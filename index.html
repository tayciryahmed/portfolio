Projects


Speech-to-text on low resource languages.

Multilingual multi-label text classification automated pipeline, from data to API.

Building parallel corpora for specialized neural machine translation: Matched multilingual documents to create a parallel in-domain corpus, using OCR, a cross-lingual version of bag-of-words and sentence alignment. Also, trained specialized neural machine translation models, using fine tuning, up-sampling and domain tags as additional features. Paper.

Data augmentation for neural machine translation: Trained translation models with state-of-the-art performance, increased the BLEU score using bilingual dictionaries, clause detection and segmentation with attention weights (Paper), back-translation and synthetic contexts, handled rare words for better translations using BPE, normalization with named-entity recognition and external knowledge integration and improved context disambiguation using pre-trained word embeddings and linguistic features: part-of-speech tags, dependency parsing labels and lemmas.

Fake news identification: The goal of this project is to develop prediction models able to identify which news is fake on www.politifact.com. The input contains of short statements of public figures (and sometimes anomymous bloggers), plus some metadata. The output is a truth level, judged by journalists at Politifact. They use six truth levels which are coded into integers to obtain an ordinal regression problem: 0: 'Pants on Fire!’, 1: 'False', 2: 'Mostly False', 3: 'Half-True', 4: 'Mostly True' and 5: 'True'.

NLP-based chatbot themed by TV series: Built a retrieval based chatbot, that incorporates personality features from a TV series. GitHub

Credit cards analysis: solved two questions related to the economic activity in France: Credit cards market share and explaining the geographical distribution of the economic sectors.

Predictive maintenance and real-time defect detection: Saved the company money and time with a machine learning solution for predictive maintenance in the automobile factories.

Gender recognition from face images: A challenging problem with applications in various knowledge domains, such as biometrics, security, surveillance and human-computer interaction.

Acoustic scene classification: Classify a test recording into one of predefined classes that characterizes the environment in which it was recorded, for example "park", "street", "office".

Credit scoring: Improve on the state of the art in credit scoring by predicting the probability that somebody will experience financial distress in the next two years.
